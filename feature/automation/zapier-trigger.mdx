**"Zapier와 연동하여 수많은 작업을 자동화할 수 있어요!🤖"** <br/>
![automation](/images/feature/automation/zapier-trigger/feature:0.png)
[Zapier](https://zapier.com/)는 어플리케이션 간의 자동화된 워크플로우를 만들어주는 도구입니다.<br/>
Zapier를 활용하여, 수많은 어플리케이션과 세일즈맵 간의 작업을 간편하게 자동화할 수 있습니다.<br/>
<br/>

본 문서에서는 세일즈맵에서 특정 웹폼을 제출하여 리드가 생성된 경우 슬랙으로 알림을 주는 예시를 설명드리겠습니다.
- 아래 예시를 따라하거나, [미리 만들어진 Zap](https://zapier.com/shared/dc66b61d7cfc5b8c7604372335457091d4e90c1d)을 활용해주세요.

## 1. 세일즈맵 Trigger 설정
- Trigger에서 App에는 세일즈맵을 선택하고, Event에는 Lead Created를 선택합니다.
![automation](/images/feature/automation/zapier-trigger/feature:1.png)
<br/>
- Salesmap의 Account 연결 과정을 진행합니다.
![automation](/images/feature/automation/zapier-trigger/feature:2.png)
- 세일즈맵 Account 연결시에 필요한 API KEY는 세일즈맵 개인설정 / 내정보 관리 / 개인 API 토큰 항목에서 발급받을 수 있습니다.
![automation](/images/feature/automation/zapier-trigger/feature:3.png)
- 리드 생성을 테스트하고 다음과 같이 리드에 webForm값이 잘 보이는지 확인합니다.
- 만약 리드 목록이 없거나 webForm값이 보이지 않는다면, 세일즈맵으로 돌아가 웹폼을 제출하여 리드를 생성후 다시 테스트합니다.
![automation](/images/feature/automation/zapier-trigger/feature:4.png)

## 2. Filter 설정
- 리드가 생성될 때마다 알림을 받고 싶지 않는 경우 필터를 설정합니다. 여기서는 특정 웹폼이 제출되어 리드가 생성된 경우만 알림을 받을 수 있도록 설정합니다.
- 웹폼의 이름으로 필터후 테스트합니다.
![automation](/images/feature/automation/zapier-trigger/feature:5.png)


## 3. 슬랙 Action 설정
- Action에서는 App에는 슬랙을 선택하고, Event에는 Send Channel Message를 선택합니다.
- Message Text에는 다음과 같이 원하는 메세지로 구성합니다.
![automation](/images/feature/automation/zapier-trigger/feature:6.png)
<br/>
- 모든 과정을 마쳤다면 Publish를 클릭하여 Zapier 워크플로우를 생성해주세요.
