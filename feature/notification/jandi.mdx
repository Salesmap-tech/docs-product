**"중요한 정보를 실시간으로 공유하여 업무 효율을 높여보세요.📢 "** <br/>

## 1. 잔디 연동. 💻
- 회사 설정 / 연동 메뉴를 클릭해주세요.
- 연동 화면에서 잔디 연동을 클릭해주세요.
![automation](/images/feature/notification/jandi/feature:0.png)
- 잔디 연동을 클릭해서 알림 설정 화면을 확인하세요.
![automation](/images/feature/notification/jandi/feature:1.png)
![automation](/images/feature/notification/jandi/feature:2.png)

## 2. 잔디 알림 설정. 💻
- 잔디에서 알림을 받고 싶은 토픽에 들어간후, 오른쪽 상단의 잔디 커넥트 버튼을 클릭해주세요.
- 연동하기 버튼을 클릭해주세요.
![automation](/images/feature/notification/jandi/feature:3.png)
- 잔디 커넥트 화면에서 Webhook 수신을 선택해주세요.
![automation](/images/feature/notification/jandi/feature:4.png)
- Webhook 수신 화면에서 프로필을 수정하고 연동 추가하기 버튼을 클릭해주세요.
![automation](/images/feature/notification/jandi/feature:5.png)
- 연동이 추가되면 Webhook URL을 복사해주세요.
![automation](/images/feature/notification/jandi/feature:6.png)
- 세일즈맵으로 돌아와 웹훅 URL 수정 버튼을 눌러 복사한 URL을 저장해주세요.
![automation](/images/feature/notification/jandi/feature:7.png)
- 웹훅 URL이 잘 저장되었는지 확인하고, 테스트 메세지를 발송해보세요.
![automation](/images/feature/notification/jandi/feature:8.png)
