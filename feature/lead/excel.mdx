**"스프레드시트 파일을 업로드하여 [리드](/dictionary/lead) 데이터를 임포트해보세요!"**

## 💻 스프레드시트 파일 업로드 화면 들어가기
- 세일즈맵의 세팅 / 데이터 가져오기 탭에서 스프레드시트 업로드 버튼을 클릭해주세요.
![excel](/images/feature/lead/excel/feature:excel_1.png)
- 딜을 생성할 파이프라인을 선택해주세요.
<Tip>만약 딜이 없다면 어떤 파이프라인을 선택하셔도 괜찮아요.</Tip>
![excel](/images/feature/lead/excel/feature:excel_2.png)

## ✏️ 스프레드시트 포맷에 맞게 작성하기
- 스프레드시트 샘플을 다운로드 받으세요. 
<Tip>샘플 데이터의 컬럼명은 수정하실 수 없습니다. 컬럼명에 맞는 데이터를 넣어주세요.<br/> People, Company, Lead, Deal 은 각각 고객, 회사, 리드, 딜을 의미합니다. 하이픈(-) 뒤에 적힌 것은 데이터의 속성을 의미합니다.</Tip>
<Tip>Lead(리드)와 Deal(딜) 업로드시 People(고객), Company(회사) 정보가 적어도 하나는 존재해야합니다.</Tip>
![excel](/images/feature/lead/excel/feature:excel_3.png)
- 포맷에 맞게 작성해주세요.
![excel](/images/feature/lead/excel/feature:excel_4.png)

## 😎 파일 업로드하고 생성하기
- 스프레드시트를 업로드해주세요.
![excel](/images/feature/lead/excel/feature:excel_5.png)
- 버튼을 클릭하여 데이터 임포트를 완료하세요. 포맷에 맞지 않아 데이터 인포트에 실패한 경우 안내 메세지에 맞게 스프레스시트를 수정해주세요.
![excel](/images/feature/lead/excel/feature:excel_6.png)
