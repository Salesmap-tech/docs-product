**"세일즈맵에서 직접 [리드](/dictionary/lead)를 생성해보세요!"**

## 1. 리드 화면 들어가기 💻
- 세일즈맵의 '리드 / 리드 수신함' 메뉴에서 리드 추가 버튼을 클릭하세요.
![create](/images/feature/lead/create/feature:create_1.png)

## 2. 리드 생성하기 ✏️
- 리드 정보를 입력하고 생성해주세요.
<Tip>리드를 생성하기 위해서는 고객 또는 회사가 필요해요. 만약 존재하지 않다면 현재 화면에서 이름을 입력하여 바로 생성할 수도 있어요.</Tip>
![create](/images/feature/lead/create/feature:create_2.png)

## 3. 생성한 리드 확인하기 😎
- 생성한 리드를 확인해주세요.
![create](/images/feature/lead/create/feature:create_3.png)
