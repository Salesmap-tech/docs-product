**"딜에서 견적서를 구성하여, 딜의 견적서와 금액을 관리하세요.📢 "** <br/>
견적서는 딜 화면에서도 만들 수 있고, 견적서 화면에서도 만들 수 있습니다. 여기서는 딜에서 바로 견적서를 구성하는 방법을 설명하겠습니다.

## 1. 견적서 구성하기. 💻
- 딜 상세화면에서 견적서를 생성해주세요.
![automation](/images/feature/quote/deal/feature:0.png)
- 생성한 견적서에 상품을 추가해주세요.
![automation](/images/feature/quote/deal/feature:1.png)
- 구독 상품의 경우 결제 횟수와 구독 시작 날자까지 입력해주세요. 예를 들어 월 구독 상품을 12개월간 구독하는 경우 12를 입력해주시고, 2023년 7월 12일에 구독을 시작하는 경우 해당 날짜를 선택해주세요.
![automation](/images/feature/quote/deal/feature:2.png)
## 2. 메인 견적서 설정하기.
- 하나의 딜에 여러개의 견적서를 설정할 수 있어요.
- 딜의 메인 견적서를 바탕으로 딜의 가격과 대시보드를 구성하게 되므로, 가장 성사될 확률이 높은 견적서를 메인 견적서로 설정해주세요.
![automation](/images/feature/quote/deal/feature:3.png)
![automation](/images/feature/quote/deal/feature:4.png)
## 3. 견적서 확인하기.
- 세일즈 / 견적 메뉴를 클릭해주세요.
- 딜에서 생성한 견적을 확인할 수 있어요.
![automation](/images/feature/quote/deal/feature:5.png)
## 4. (선택) 견적서와 상품 사이의 커스텀 필드 사용하기.
- 상품의 필드는 아니지만, 상품을 견적서에 등록할 때 필요한 경우라면 견적서-상품 데이터 필드를 이용해보세요.
- 예를 들어 배너 광고 상품의 경우 타게팅 지역은 상품에서 정의할 수 없지만, 딜의 견적서에 입력 할때는 필요해요.
- 세일즈맵의 '회사 설정 / 데이터 필드 관리' 메뉴에서 상품 탭을 클릭해 견적서-상품 데이터 필드를 생성하세요. 이때 컬럼 조회 유무 값을 모두 체크해주세요.
![automation](/images/feature/quote/deal/feature:6.png)
- 딜의 견적서를 등록하는 다이얼로그에서 타겟팅 지역을 확인하고 값을 입력해주세요.
![automation](/images/feature/quote/deal/feature:7.png)
- 입력한 값이 견적서에서 잘 조회되는지 확인해주세요.
![automation](/images/feature/quote/deal/feature:8.png)
