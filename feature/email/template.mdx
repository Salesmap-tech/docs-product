**"이메일 템플릿으로 효율적인 이메일을 업무를 경험해보세요!📨 "** <br/>
![automation](/images/feature/email/template/feature:0.webp)
자주사용 되는 이메일 템플릿을 저장할 수 있는 기능입니다. <br/>
드래그-앤-드랍 형태로 템플릿을 만들거나, 줄글 형태의 템플릿을 만들 수 있습니다.<br/>

## 1. 이메일 템플릿 사용의 이점
1. 개별화된 이메일 작성: 이메일 템플릿에는 [개인화된 변수](/feature/email/variable)(예: 고객 이름, 회사명 등)를 포함할 수 있습니다.
2. 표준화된 커뮤니케이션: 기업 전체에서 동일한 메시지와 정보를 전달할 수 있습니다. 이로 인해 고객과의 커뮤니케이션에서 혼선이 줄어듭니다.
3. 일관성: 모든 이메일의 스타일과 톤을 통일하여 작성할 수 있습니다. 이를 통해 기업의 브랜드 이미지를 향상시키고, 전문적인 인상을 줄 수 있습니다.

<Tip>세일즈맵에서 시나리오별로 정의된 템플릿을 사용할 수 있습니다.</Tip>

## 2. 템플릿 생성하기 💻
- 세일즈맵의 '이메일 / 템플릿' 메뉴에서 템플릿 추가 버튼을 클릭하세요.
- 처음부터 시퀀스를 만들기 어려운 경우에는 사전에 정의된 템플릿을 수정하여 사용해보세요.

## 3. 드래그-앤-드랍 템플릿 & 줄글 템플릿 💻
- 목적에 맞는 템플릿을 사용해보세요.
- <code>드래그-앤-드랍</code> : 시각적으로 매력적인 이메일 디자인이 필요한 경우에 적합합니다. 이미지, 버튼, 텍스트 등 다양한 요소를 활용해 보세요.
![automation](/images/feature/email/template/feature:1.webp)
- <code>줄글</code> : 간결하고 명확한 메시지 전달에 초점을 맞춘 이메일을 작성할 수 있습니다.
![automation](/images/feature/email/template/feature:2.webp)

## 4. 템플릿 사용하기 📍
- 생성된 템플릿은 세일즈맵에서 여러 용도로 사용할 수 있어요.
1. [이메일 마케팅](/feature/automation/group) : 리드 그룹에 이메일을 일괄 발송하고 성과를 확인해보세요.
2. [시퀀스](/feature/automation/sequence) : 여러가지 연속된 작업을 순서대로 배치하고 자동화해보세요.
3. [워크플로우](/feature/automation/workflow) : 특정 이벤트를 트리거로 사용하여, 후속 작업들이 자동 실행되도록 해보세요.
