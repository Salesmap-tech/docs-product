title: "커스텀 데이터 필드"

**"매출로 이어지는 고객들은 어떤 특징이 있나요? 우리 회사만의 데이터 필드를 정의해보세요.👆"**

## 1. 데이터 필드 확인하기 💻
- 세일즈맵의 '회사 설정 / 데이터 필드 관리' 메뉴를 클릭해주세요.
- 리드/딜, 고객, 회사, 견적서, 상품의 기본적으로 설정된 데이터 필드를 확인하세요.
  ![customfield](/images/setting/customfield/setting:customfield_1.png)

## 2. 커스텀 데이터 필드 생성하기 📘
- 커스텀 데이터 필드를 생성할 대상을 탭에서 선택해주세요.
- 유형은 텍스트, 숫자, 날짜, 단일 선택, 사용자, 회사, 고객이 있어요. 이중 원하는 데이터 유형을 선택해 주세요.
<Tip>
  사용자는 현재 워크스페이스에 초대된 사용자, 회사는 세일즈맵에 등록한 회사, 고객은 세일즈맵에 등록한 고객을 의미해요.
</Tip>
- 예를 들어 리드/딜에서 단일 선택 데이터 유형을 사용하면, 딜의 긴급도를 몇 가지 유형으로 나누어 관리할 수 있어요.
  ![customfield](/images/setting/customfield/setting:customfield_2.png)

## 3. 데이터 필드 표시 유무 설정하기
- 커스텀 데이터를 상세 화면 또는 생성 다이얼로그에서 표시할지 말지 설정 할 수 있아요.
- 상세 화면을 체크하면 상세 화면에서 필드 조회 및 수정이 가능하고, 생성 다이얼로그를 체크하면 대상을 생성하는 다이얼로그에서 조회 및 수정이 가능해요.
  ![customfield](/images/setting/customfield/setting:customfield_3.png)

## 4. 커스텀 데이터 필드 순서 변경하기
- 상세 화면 또는 생성 다이얼로그에서 커스텀 필드의 순서를 변경할 수 있어요.
- 표의 오른쪽 끝 톱니바퀴 아이콘을 선택하고, 드래그 앤 드랍으로 순서를 변경하세요.
  ![customfield](/images/setting/customfield/setting:customfield_4.png)
