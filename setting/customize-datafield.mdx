**"매출로 이어지는 고객들은 어떤 특징이 있나요? 우리 회사만의 데이터 필드를 정의해보세요.👆"**

## 1. 커스텀 데이터 필드 생성가기 💻
- '세팅 / 커스텀 데이터 필드'메뉴를 클릭해주세요.
  ![customfield](/images/setting/customfield/setting:customfield_1.png)


## 2. 커스텀 필드 데이터 유형 📘
- 텍스트, 숫자, 날짜, 단일 선택 중 원하는 데이터 유형을 선택해 주세요.
- 단일 선택 데이터 유형을 사용하면, 고객을 몇 가지 유형으로 나누어 관리할 수 있습니다.
- 예를 들어, 기업 규모(대기업, 중견기업, 중소기업)에 따라 분류할 수 있어요.
  ![customfield](/images/setting/customfield/setting:customfield_2.png)


## 3. 생성된 커스텀 필드 확인하기 😎

- 생성된 커스텀 필드는 아래와 같이 확인하실 수 있습니다.
